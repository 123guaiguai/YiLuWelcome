markdown

⭐什么是markdown

​ Markdown是一种轻量级标记语言，它允许我们使用简单易读的纯文本格式编写文档，再转换为有效的HTML或者XHTML文档，能够很好地满足我们轻写作以及撰写论坛文本的需求。

🎈为什么是markdown

  1. 简洁——不着一色，而形神俱备

    想必各位同学进入大学以来，肯定用过Word文本编辑软件，突然换用markdown类文本编辑软件会觉得功能太过简单，但这正是其可贵之处。与艺术追求留白，程序代码追求简洁易懂一样，简洁的文本会让你忽略字体样式、段落间距等某些情况下的次要因素，更加专注于文章内容和逻辑架构。

  2. 效率——向来枉费推移力，此日中流自在行

    操作上，由于功能简洁，因此相应的快捷键也不多，熟练的markdown 撰写者往往熟悉大部分常用的快捷键，对于笔记本生产来说，减少鼠标的使用会让我们的工作学习更便捷、更迅速。

    由于markdown本身的特性，其文本转换为html文件几乎不发生变化，因此对于我们撰写网页文本、论坛文本都有很大的帮助。

😄关于该网站

  • 由于网站架设在外网，所以最好挂上梯子进行访问，不然的话可能每次刷新你都要点击一下 RELOAD 按钮，同样图片的加载可能会慢一点。

  • 我们在后面可能对网站题目等实时更新，到时候会有相应的提示~

💖一些学习的小建议

  • 在这之前有个学弟和我聊的时候提到这么一个问题，他说在开学时为了努力完成工作室的题目，自己C语言和微积分线代都学的很差。因此我们想说，工作室的题目仅仅应该是你学习之余完成的一个项目，更重要的是你应该把课内的知识学好，因为这是基础，如果你不学课内的C语言，数学基础和计组计网操作系统这些核心课程而是直接一来就Java，python，html,点击了解-> 千锋教育培训机构 ，北大青鸟职业教育

  • 因此，在这为期近一个月的招新过程中，更重要的是享受学习的过程而不是一味地注重结果，我们为你准备了丰富的基础知识题库，有趣的C语言题目以及前后端方向题。在你学习之余了解了解前后端技术，也为自己的兴趣开辟了一条新的道路，就算最后没能加入工作室，相信你也能从中获得很多知识。并且在下学期和大二的工作室招新中更能游刃有余。

😜我该按照什么路线去完成

在我们的左侧菜单栏中分为四部分:

  1. 基础知识 这一部分是对你最有用的作业部分，上面的很多技巧老师不会教但是应该是你以后学习工作时的必备技能，同时在完成这些内容时你也会在各个方面得到提高。

  2. C语言 你可以巩固下C语言的知识，这里有一些有趣的题目尝试着看看。

  3. 前端 如果你对前端更感兴趣，不妨来试试这些题目，题目难度按照你的学习进度依次推进，不会太难的，放心。

  4. 后端 同样如果你对后端感兴趣，来看看这里，学习过程中试着完成题目以巩固你的学习成果。

  5. YILU STUDIO 资源共享平台 知识无界限，这里有一些学长学姐们精心撰写或挑选的知识文章，你可以学习参考（不过由于水平有限如有错误还请更正啦），当然如果你有好的文章也欢迎联系我们 2050406742@qq.com ，我们可以发布上去并注明你的作者信息。

学习过程中，你可以把 基础知识 + C语言 + 前端或后端 三个部分同步进行，尤其是我们为你精心准备的C语言和基础知识题目，无论你是否有意愿加入工作室，好好的去看看这部分题目都会对你有帮助，加油哇！

基础知识部分和C语言部分我们建议大家都去做一做，虽然C语言可能很难，然后前后端的题目你可以选择你感兴趣的方向完成。

我们的招新不规定特定的题目数量，所以放手去做吧！
