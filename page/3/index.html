Git

Git在日常开发以及未来的协作开发中，git是必不可少的工具，所以，提前了解，使用，以及熟练掌握git是非常有必要的，本文不会过多介绍git的细节用法，而是引导大家如何去学习，使用git

什么是git

git是一种分布式版本管理系统，你可以想象，当一个开发团队同时开发一个项目，由于每个人开发的模块不同，最后进行模块对接时，肯定少不了代码合并，或者说由于开发同一个板块产生了冲突，需要解决冲突。如果没有一种手段或工具去对每个人开发的模块进行管理，那么在解决上述问题（当然肯定不止上述问题）时，会变得非常复杂，耗时耗精力。

所以我们需要一个工具去对我们的项目代码进行管理，无论是开发时，还是在维护时（比如debug），git便可以做这件事。

git可以帮助我们做代码管理，在合并时，会自动做代码对比检测，如果有冲突，也能够快速方便地解决冲突。

配置vscode

工欲善其事，必先利其器。想要优雅且高效的编写代码，必须熟练使用一款开发工具。

vscode简介

VSCode是一款微软出的轻量级编辑器，它本身只是一款文本编辑器而已，所有的功能都是以插件扩展的形式所存在，想用什么功能就安装对应的扩展即可，非常方便，同时也支持非常多的主题和图标，外观比较好看，重要的是VSCode支持各大主流操作系统，包括Windows、Linux和Mac OS。所以就选择它作为自己的一款主要的编辑器来使用。

它和我用的devcpp（devc++）有什么不同

不同点在于vscode仅仅是一款编辑器，所以你并不能直接开始编译运行C语言，而要经过一系列的配置，而devcpp是一款集成开发环境(IDE),因此可以下载即用，对新手友好，而vscode扩展丰富，界面美观。

markdown

⭐什么是markdown

​ Markdown是一种轻量级标记语言，它允许我们使用简单易读的纯文本格式编写文档，再转换为有效的HTML或者XHTML文档，能够很好地满足我们轻写作以及撰写论坛文本的需求。

🎈为什么是markdown

  1. 简洁——不着一色，而形神俱备

    想必各位同学进入大学以来，肯定用过Word文本编辑软件，突然换用markdown类文本编辑软件会觉得功能太过简单，但这正是其可贵之处。与艺术追求留白，程序代码追求简洁易懂一样，简洁的文本会让你忽略字体样式、段落间距等某些情况下的次要因素，更加专注于文章内容和逻辑架构。

  2. 效率——向来枉费推移力，此日中流自在行

    操作上，由于功能简洁，因此相应的快捷键也不多，熟练的markdown 撰写者往往熟悉大部分常用的快捷键，对于笔记本生产来说，减少鼠标的使用会让我们的工作学习更便捷、更迅速。

    由于markdown本身的特性，其文本转换为html文件几乎不发生变化，因此对于我们撰写网页文本、论坛文本都有很大的帮助。
